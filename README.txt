Encoding - une classe permettant de convertir des données dans différents formats

Cette classe utilise la classe BigInteger du package java.math


Méthodes (statiques) disponibles :


String hexToString(String hex)
String asciiToString(int[] ascii)
String intToString(BigInteger i)

String stringToHex(String str)
String asciiToHex(int[] ascii)
String intToHex(BigInteger i)

int[] stringToASCII(String str)
int[] hexToASCII(String hex)
int[] intToASCII(BigInteger i)

BigInteger stringToInt(String str)
BigInteger hexToInt(String hex)
BigInteger asciiToInt(int[] ascii)
